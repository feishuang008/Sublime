# sublime
settings of sublime

## 包含以下默认配置

### 配置列表

| NAME           | DESCRIBE |
| :--            | :-- |
| docblockr      | 代码注释自动生成 |
| babel          | ES6和JSX语法高亮 |
| babel Snippets | ES6和React代码段 |
| emmet          | HTML和CSS快捷编写，同时会安装pyV8 |
| JSFormat       | JS格式化插件 |
| stylus         | Stylus语法高亮 |

说明：
- Babel功能需要安装Node和Babel，快捷键为CTRL+B。
- 还安装了CSS展开与折叠的功能
